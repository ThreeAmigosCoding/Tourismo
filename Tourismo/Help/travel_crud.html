Travel Details overview

In travel details overview you are able to make changes for existing travel or to create a new one. That depends on the button that you have clicked in the Travels Overview Page

Name, Description and Image

At the top, there are input fields for name and description and a button for travel image upload.

List Views and Search Bar

The window features three list views: "All Attractions," "Default Attractions," and "Additional Attractions." These list views provide you with a convenient way to manage tourist attractions for your travel itinerary. You can drag and drop attractions between the list views to organize them according to your preferences. Additionally, there is a search bar available in the "All Attractions" list view to help you quickly find specific attractions.

Managing Attractions

1. All Attractions: This is the main list view that displays all the tourist attractions available in the system. You can browse through the list to find attractions that interest you. The search bar located at the top of this list view allows you to search for attractions by their names. Simply enter your search query and the list will update in real-time to display the matching attractions.

2. Default Attractions: This list view contains the attractions that are automatically included in a new travel itinerary. You can drag attractions from the "All Attractions" list view and drop them into this list to add them to your travel plan. To remove an attraction from the default list, simply drag it back to the "All Attractions" list view.

3. Additional Attractions: This list view is meant for attractions that you want to add to your travel itinerary beyond the default ones. Similarly to the "Default Attractions" list view, you can drag attractions from the "All Attractions" list view and drop them into this list to include them in your travel plan. If you decide to remove an attraction from the additional list, drag it back to the "All Attractions" list view.

Using the Search Bar

The search bar in the "All Attractions" list view allows you to quickly find specific attractions based on their names. To search for an attraction, follow these steps:

  1. Click on the search bar at the top of the "All Attractions" list view.
  2. Enter the name of the attraction you want to find.
  3. As you type, the list view will update in real-time to display the attractions that match your search query.
  4. To clear the search results and display all attractions again, remove the text from the search bar.

Accommodation and Periods

Accommodation:

  1. In the accommodations list view, browse through the available options. The list view displays the existing accommodations.
  2. Select the desired accommodation by clicking on it. The selected accommodation will be displayed.
  3. The chosen accommodation will be associated with your travel.

Periods:

  1. Select the start date of the travel period using the date picker. Click on the input field to open the date picker, and choose the desired date.
  2. Select the end date of the travel period using the date picker. Similarly, click on the input field to open the date picker, and choose the desired date.
  3. Once both start and end dates are correctly inputted, the "Add Period" button will become available.
  4. Click on the "Add Period" button to add the travel period to the list view below.
  5. If you want to remove a specific travel period, select the period from the list view, and the "Remove Period" button will become available.
  6. Click on the "Remove Period" button to delete the selected travel period from the list view.

If all required fields are filled correctly, the "Save" button will become available.

Additionally, if you are in "Update" mode, "Delete" button will show in the bottom right corner.
